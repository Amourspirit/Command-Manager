Adding Componets To Toolbar

  1. AddingTab to Visual Studio Toolbox
  2. Create Command Manager Tab
  3. Choose Items Menu
  4. Browse to CommandManager.dll
  5. Select CommandManager.dll
  6. Filter using CommandManger and choose CommandManager Items.
  7. Components are now added to Visual Studio Toolbar.
